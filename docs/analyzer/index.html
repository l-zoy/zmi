分析包大小

介绍#

使用 webpack-bundle-analyzer 分析 JavaScript 包。 这有助于你了解代码膨胀的来源。

默认已添加,通过设置环境变量来开启,比如:

$ ANALYZE=1 zmi dev
// or
$ ANALYZE=1 zmi build

或者在.env文件中定义

PORT=3000
ANALYZE=1
