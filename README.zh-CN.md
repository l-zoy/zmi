## Features

- 🔩 **可配置的** Zmi 完全是可配置的。

- 📦 **开箱即用** 一个依赖项即可同时支持 `vue` / `react` / `miniapp`. 默认支持 `typescript`

- 🙅 **大量不自研** 优先使用成熟的社区解决方案, 保证稳定性

- 🤷‍♂️ **没有路由** 不参与 runtime

- 🎉 基于最新的 **webpack5** 支持 vue3.x, react 17.x, 小程序 file to file 编译
